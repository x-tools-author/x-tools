# xTools

{: .note }
> xTools始于2018...

*xTools*(曾用名：QtSwissArmyKnife，瑞士军刀，QSAK)是一款开始于2018年基于Qt开发的开源、免费，跨平台、多功能的开发调试工具，支持串口，蓝牙，UDP，TCP, Web Socket, Modbus，CAN Bus和CoAP等调试。*xTools*旨在为开发者提供一个强大的调试平台，简化各种通信协议的调试过程，提高开发效率。

> *xTools后续会推出专业版本（xTools Pro），专业版本付费提供，专业版本特有功能不开源。目前微软商店上架的专业版与社区版本没有功能差别。*

*xTools*是系列软件的统称，包括xTools，xBLE, xModbus, xCANBus, xMQTT, xCoAP, xOPCUA, xHttp，xJT808和xFlow。
